Clone repository:
```bash
$ mkdir CSE110_ROOMI
$ cd CSE110_ROOMI
$ git clone https://github.com/lambrendan/roomi.git
```
Install Node
```bash
$ curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.0/install.sh | bash
$ nvm install 10.12
$ nvm use 10.12
```
How to run application:
```bash
$ node server.js
```
check localhost:3000 to our application!
